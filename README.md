# Gaurav Kumar Singh - AI-for-All

Static personal website ready for:

- GitHub Pages
- Railway

## Local preview

Open `index.html` directly, or run a static server in this folder.

## GitHub

```powershell
git init
git branch -M main
git remote add origin https://github.com/gorky117-cell/gaurav_personal.git
git add .
git commit -m "Initial website import"
git push -u origin main
```

## GitHub Pages

Because this is a static site, you can enable Pages from the `main` branch and `/ (root)` folder in repository settings.

## Railway

This repo includes a `Dockerfile` and `nginx.conf`, so Railway can deploy it directly as a static site container.
